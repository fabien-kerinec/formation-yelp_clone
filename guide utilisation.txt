------------------ EXECUTION EN LOCAL ------------------------
Etant donnée que c'est une application react et nodeJS
Il faut avoir node et npm d'installer
ICi j'utilise node 8.4.0 et npm 5.3.0
Par soucis de taille de fichier, j'ai retiré le dossier "node_module" qui permet d'installer les differents packages.
Pour les installer dans le dossier principal (P1 ; P2 ; P3/front ; P3/API) faire un "npm install" dans une console.
Puis pour lancer l'application : "npm start"

Le front se lance sur le localhost:3000
Dans le code (P3/src/component/content ligne 67) est indiqué l'addresse de la requète pour la donnée.
De base elle indique l'addresse de mon serveur VPS sur le port 3006.
En local ça ne posera pas de soucis particulier (dont je parlais plus haut) car le localhost n'est pas détécté en https.

Si jamais vous voulez tester avec l'api "local" vous pouvez lancer l'api avec un "npm start"
Le serveur se lancera sur le "localhost:3006"
Il faudra evidemment changer l'adresse a la ligne cité plus haut
